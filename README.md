## typedoc-plugin-explicit-include

### What

A plugin for [Typedoc](http://typedoc.org)

Use an annotation (in a comment) to set code to "External". All other reflections will not be exported

### Examples

```
/**
 * A function
 * @include
 */
function someFunction() {

}
```
